## Installation
1. Add jitpack repository in pom file
    ```bash
    <repositories>
     <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
     </repository>
    </repositories>

2. Add hosp-ai dependency (check latest version)
    ```bash
    <dependency>
     <groupId>com.github.r7b7</groupId>
     <artifactId>hosp-ai</artifactId>
     <version>v1.0.0-alpha.2</version>
    </dependency>
